# My Portfolio API

## Project summary

API to serve data, from my GitHub repositories, to my [`my-portfolio-app`](https://github.com/Massimilianok/my-portfolio-app) project.

## Main technologies

- JavaScript
- Node.js

## Main packages

- express
- node-fetch
- dotenv
- cors
- express-validator
- nodemailer

## Third party API

- GitHub API

## SMTP service

- Mailtrap

## Hosting

- Heroku
